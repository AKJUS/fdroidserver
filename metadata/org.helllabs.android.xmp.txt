Summary:Tracker module player
Description:
The Extended Module Player, or xmp, is a portable module player. Xmp plays
over 90 mainstream and obscure module formats from Amiga, Atari, Acorn,
Apple IIgs and PC. This is an experimental port built with the Android NDK
and a Java interface. All module formats are supported, but external file
compressors are not.
